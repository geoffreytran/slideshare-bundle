ZymSlideShareBundle
===================

This bundle is a wrapper for Zend Framework 2.0's `Zend\Service\SlideShare` 
component. It exposes the api through the service `zym_slide_share.api`.

Configuration
-------------

Only api_key and shared_secret are required.

	zym_slide_share:
		api_key:
		shared_secret:
		username:
		password:
